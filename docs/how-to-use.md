# How to Use

## React

### Installation - React

```sh
npm i meistericons-react --save
```

### Usage

```jsx
import {Add} from 'meistericons-react'

export default ()=>{
    return <Add/>
}
```

## Vue

### Installation - Vue

```sh
npm install meistericons-vue --save-dev
```

### Usage - Vue

```jsx
<script>
import {ArrowDown} from "meistericons-vue"


export default {
    name:'My Component',
    components: {ArrowDown}
}
</script>

<template>
    <ArrowDown/>
</template>
```

## Vue3
