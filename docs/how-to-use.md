# How to Use

## React

### Installation - React

```npm
npm i meistericons-react --save
```

### Usage

```jsx|tsx
import {Add} from 'meistericons-react'

export default ()=>{
    return <Add/>
}
```

## Vue

### Installation - Vue

```npm

npm install meistericons-vue --save-dev
```

### Usage - Vue

```jsx|tsx

<script>
import {ArrowDown} from "meistericons-vue"


export default {
    name:'My Component',
    components: {ArrowDown}
}
</script>

<template>
    <ArrowDown/>
</template>
```

## Vue3

### Installation - Vue3

```npm

npm install meistericons-vue-latest --save-dev
```

### Usage - Vue 3
